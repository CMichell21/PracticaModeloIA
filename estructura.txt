	Jaybedeapi: Permite conectar código python a base de datos mediante java jdbc, hace uso de la librería Jpyp, permite compatibilidad con DB-API es una especificación que define una interfaz estándar para acceder a bases de datos desde Python.

	Jpype: Modulo de Python que proporciona acceso a java desde Python es un puente entre ambos lenguajes, esto lo logra conectando Python y java a nivel nativo mediante la interfaz nativa de java, para ser utilizada debe iniciar primero el java virtual Machine.
